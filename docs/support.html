Support

Getting Help

If you're experiencing issues with Lift or have questions, here's how to get help:

Report a Bug

Found a bug? Please report it by opening an issue on our GitHub repository with a description of the problem and steps to reproduce it.

Feature Requests

Have an idea to make Lift better? We'd love to hear it! Open an issue on GitHub with your suggestion.

Account Issues

For account-related issues such as data deletion requests or account recovery, please contact us through GitHub.

FAQ

How do I sync my data across devices?
Your data automatically syncs when you're signed in with the same account on multiple devices.

Can I export my workout data?
Data export functionality is available in the app settings.

Is my data backed up?
Yes, your data is securely stored in the cloud and backed up automatically.
